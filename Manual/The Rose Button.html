On the Subject of The Rose Button

This is a button. It is rose.

Holding the rose button starts a stopwatch which stops when the button is released. Stop the stopwatch as close to exactly one second without going over.

1 second exactly is a home run, which solves the module instantly. 0.97 seconds is a single, and 0.98 and 0.99 are a double and a triple, which advance the runner that many bases. Get the runner past 3rd base to solve the module.

Any other time is an out. Getting three outs will cause a strike and reset the module.
